# 📊 沪深300智能选股系统 - 开发指南

## 项目概述

这是一个基于技术指标和动量策略的A股自动化选股与回测系统，专注于沪深300成分股的量化分析。系统采用Python开发，使用腾讯财经API获取实时数据，实现智能筛选、基本面分析、回测验证和邮件推送等功能。

### 核心特性

- **智能筛选** - 基于PE、PB、ROE、PEG等多维度基本面+技术面筛选
- **基本面分析** - ROE(股本回报率)、利润增长率、股息率等核心指标
- **动量策略** - 20日动量计算，捕捉短期趋势
- **换手率评分** - 基于换手率评估流动性，消除市值影响，对大小盘股更公平
- **5维评分系统** - 技术面(30分)+估值(25分)+盈利能力(30分)+安全性(10分)+股息(5分)
- **分项得分显示** - 详细展示每只股票的技术面、估值、盈利能力、安全性、股息等各项得分
- **历史回测** - 支持单日/多日回测，验证策略有效性
- **邮件推送** - 每日盘后自动发送选股结果(含ROE等基本面数据)
- **智能缓存** - 加速数据获取，支持离线分析
- **腾讯财经API** - 100%成功率，稳定可靠的数据源

## 核心模块说明

### 1. 数据获取模块 (`src/data/data_fetcher.py`)

- 使用腾讯财经API获取实时股票数据
- 实现智能重试机制，包含指数退避和随机延迟
- 批量获取数据以提高效率
- 带有智能缓存机制，减少重复请求
- 计算PEG(市盈率相对盈利增长比率)指标用于估值评估

### 2. 股票筛选模块 (`src/analysis/stock_filter.py`)

- 实现5维评分系统
- 根据PE、PB、ROE、动量等多维度筛选
- 基于换手率评估流动性
- 按行业分组，每个行业最多选择一只股票

### 3. 市场分析模块 (`src/analysis/market_analyzer.py`)

- 综合数据获取和筛选模块
- 执行每日盘后分析
- 生成详细报告
- 保存分析结果

### 4. 邮件通知模块 (`src/notification/email_sender.py`)

- 生成专业级HTML格式邮件
- 包含详细的股票分析数据
- 支持Markdown附件
- 完整的错误处理机制
- 在输出表格中包含PEG(市盈率相对盈利增长比率)估值指标

### 5. 任务调度模块 (`src/scheduler/task_scheduler.py`)

- 基于`schedule`库的智能调度
- 每日16:00执行盘后分析
- 每日08:30发送邮件报告
- 完整的任务历史记录和性能统计

## 评分系统详解

### 5维综合评分系统（满分100分）

```
综合分数 = 技术面(30) + 估值(25) + 盈利能力(30) + 安全性(10) + 股息(5)

# 1. 技术面 (30分)
## 涨跌幅 (10分)
- 当日涨幅 >5%   → 10分
- 当日涨幅 >2%   → 7分
- 当日涨幅 >0%   → 4分

## 动量 (15分)
- 20日动量 >15%  → 15分
- 20日动量 >10%  → 12分
- 20日动量 >5%   → 8分

## 流动性 (5分) - v3.1优化: 改用换手率
- 换手率 1-3%    → 5分 (最佳：流动性适中)
- 换手率 3-5%    → 4分 (良好)
- 换手率 5-8%    → 3分 (充足但略偏高)
- 换手率 0.5-1%  → 2分 (偏低)
- 换手率 >8%     → 1分 (投机性强，风险高)

# 2. 估值面 (25分) - v4.4优化
- PE <10         → 10分 (低估值)
- PE <20         → 7分 (合理估值)
- PB <2          → 10分 (低估)
- PB <4          → 8分 (合理)
- PB <7          → 5分 (适中)
- PEG <1         → 5分 (优质成长股)

# 3. 盈利能力 (30分) - 核心
- ROE >20%       → 15分 (巴菲特标准)
- ROE >15%       → 12分
- 利润增长 >30%  → 15分
- 利润增长 >20%  → 12分

# 4. 安全性 (10分) - v4.4优化
- PB市净率 <1.0  → 3分 (破净，极度安全)
- PB市净率 <1.5  → 2分 (接近破净)
- PB市净率 <2.5  → 1分 (有安全边际)
- 股息率 >5%     → 3分 (高分红，经营稳定)
- 股息率 >3%     → 2分
- 换手率 <2%     → 4分 (筹码稳定，非投机股)
- 换手率 <5%     → 3分

# 5. 股息 (5分) - v3.1优化: 支持手动修正
- 股息率 >5%     → 5分
- 股息率 >3%     → 4分
- 股息率 >2%     → 2分
- 股息率 >0%     → 1分

评级标准:
A+: 90-100分  A: 80-89分  B+: 70-79分
B: 60-69分    C: 50-59分  D: <50分
```

### ROE计算方法

本系统采用创新的ROE估算公式:

```
ROE = (PB / PE) × 100

# 原理:
# ROE = 净利润/净资产
# PB = 市值/净资产, PE = 市值/净利润
# 因此: PB/PE = 净利润/净资产 = ROE

# 验证实例:
贵州茅台: PB=13.95, PE=36.00 → ROE=38.7% ✓
平安银行: PB=0.59, PE=4.61  → ROE=12.8% ✓
```
